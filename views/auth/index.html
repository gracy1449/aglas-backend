How It Works

Simple. Fast. Accurate.

Four steps from assignment creation to instant graded results.

1

Lecturer Creates Assessment

Set questions, define answer keys, assign marks and publish to enrolled students.

2

Student Enrolls & Attempts

Students log in, enroll in courses and take assessments within the time limit.

3

AGLAS Auto-Grades

The system compares answers, matches keywords and executes code — all in seconds.

4

Instant Results & Feedback

Scores are displayed immediately with per-question feedback for improvement.

Assessment Types

Three Ways to Grade

Every type of academic assessment, handled automatically.

🎯

Objective

Multiple-choice and true/false questions graded instantly with exact-match accuracy.

  • Multiple Choice Questions (MCQ)
  • True / False Questions
  • Instant exact-match grading
  • Per-question correct answer shown
📝

Theory

Long-form theory answers evaluated using lecturer-defined keyword sets and weighted scoring.

  • Keyword-based matching engine
  • Partial marks for partial answers
  • Keyword hit report in feedback
  • Configurable by lecturer
💻

Programming

Code submissions run against test cases in a sandboxed environment, marks awarded per passing test.

  • Test-case execution engine
  • Partial credit for partial solutions
  • Supports multiple languages
  • Safe sandboxed execution
Core Features

Everything You Need

Built for Nigerian universities, designed for everyone.

Instant Grading

Submissions are graded the moment a student clicks submit — no waiting, no manual marking.

Countdown Timer

Each exam has a live countdown timer. When time runs out, the exam auto-submits automatically.

Secure & Reliable

Encrypted passwords, session-based authentication and protected routes keep your data safe.

Grades & Analytics

Students track scores and progress. Lecturers view class performance and submission rates.

Role-Based Access

Detailed Feedback

Students receive per-question feedback, keyword hit reports and score breakdowns after submission.
